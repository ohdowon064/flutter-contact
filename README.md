## 플러터 앱 개발방식
- 위젯 짜집기
- 모든 것이 위젯
- 기본 위젯
  - 글자위젯: Text("텍스트")
  - 이미지위젯: Image.asset("경로")
  - 아이콘위젯: Icon(Icons.아이콘이름)
    - 이미지는 pubspec.yaml에 등록 필요
  - 박스위젯: Container(), SizedBox()
- flutter에서 단위는 lp
  - 50lp == 1.2cm
- Center(): 자식 위젯의 기준점을 중앙으로 설정
